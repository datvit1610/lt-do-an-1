Firebase Web Push Notification Test

Đây là một trang test đơn giản để hiển thị cách nhận và hiển thị thông báo Push từ Backend của bạn.

Kiểm tra trạng thái quyền...
FCM Token của bạn sẽ hiển thị ở đây:
(Chưa có token)

Sau khi có **FCM Token**, hãy sao chép toàn bộ chuỗi token đó. Sử dụng nó trong request POST từ Postman (hoặc công cụ tương tự) tới API Spring Boot của bạn (`POST http://localhost:8080/api/v1/notification`) với body JSON như sau: 

{
  "recipientToken": "<DÁN_FCM_TOKEN_CỦA_BẠN_VÀO_ĐÂY>",
  "title": "Thông báo Test!",
  "body": "Nội dung thông báo realtime.",
  "image": "https://cdn-icons-png.flaticon.com/512/1051/1051930.png",
  "data": {
    "key1": "value1",
    "key2": "value2"
  }
}
